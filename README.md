
# 🛒 SmartMarket AI – Intelligent Supermarket Monitoring System

An AI-powered smart supermarket monitoring and cashier simulation system designed to help store managers monitor inventory movement, detect product transactions using webcam vision, and generate automatic daily performance reports.

This project demonstrates how **Computer Vision + Smart Inventory Logic + AI Monitoring Dashboard** can be integrated into a modern retail environment.

---

## 🚀 Project Overview

SmartMarket AI is an intelligent retail monitoring demo system that simulates real-time supermarket operations such as:

- 📦 Warehouse stock monitoring  
- 🛍️ Display shelf monitoring  
- 🧠 AI product detection using webcam  
- 💰 Smart cashier transaction simulation  
- 📊 Sales analytics dashboard  
- 📑 Automated daily closing report  

This demo is designed for **AI Portfolio Showcase & Concept Validation**.

---

## 🖥️ Main Features

### 📊 Dashboard Overview

The dashboard provides a real-time summary of supermarket activity including:

- Total stock in warehouse  
- Total stock in display area  
- Today's revenue  
- Today's profit  
- Sales per item visualization (bar chart)  
- Latest transaction activities  

This page helps store managers quickly understand store performance.

---

### 🎥 AI Monitoring System

The system simulates intelligent monitoring using a webcam.

It can detect:

- Product movement into warehouse  
- Product movement into display zone  
- Consumer presence in supermarket area  

This enables monitoring of **inventory flow and store activity in real-time.**

---

### 📦 Inventory Management

Smart inventory module allows:

- Adding new products using webcam photo capture  
- Editing product data (price, stock quantity, barcode, etc.)  
- Moving stock from warehouse → display area  
- Automatic update of:
  - Remaining stock  
  - Display quantity  
  - Sold quantity  

Inventory status is dynamically updated after each cashier transaction.

---

### 🧾 Smart Cashier System

The cashier page provides AI-assisted transaction simulation:

- Detect product via webcam capture  
- Automatically retrieve product information  
- Show price & transaction total  
- After payment:
  - Sold quantity increases
  - Stock balance updates automatically  

This simulates **vision-based POS automation.**

---

### 📑 Daily Closing Report

At the end of the day the system generates a summarized performance report including:

- Total revenue  
- Total profit  
- Total items sold  
- Per-item profit breakdown  

Features:

- Export report as PDF  
- Printable closing report  

This helps managers track operational performance efficiently.
